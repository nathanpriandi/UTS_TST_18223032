### Prerequisites

This project is self-contained and does not require a separate backend server. You only need Node.js and Yarn installed.

# Overview

This starter template is built with:

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Typescript](https://www.typescriptlang.org/)
- [Fake Store API](https://fakestoreapi.com/) for product and category data.

Features include:

- Full ecommerce support:
  - Product Detail Page
  - Product Overview Page
  - Product Categories
- Full Next.js support:
  - App Router
  - Next fetching/caching
  - Server Components
  - Server Actions
  - Streaming
  - Static Pre-Rendering

**Note:** User accounts, cart, and checkout functionality have been disabled as they are not supported by the Fake Store API.

# Quickstart

### Install dependencies

Use Yarn to install all dependencies.

```shell
yarn
```

### Start developing

You are now ready to start up your project.

```shell
yarn dev
```

### Open the code and start customizing

Your site is now running at http://localhost:8000!
